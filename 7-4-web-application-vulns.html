A01:2021 - BROKEN ACCESS CONTROL
РћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђРћђ
Рђб Users acting outside intended permissions
Рђб IDOR (Insecure Direct Object References)
Рђб Missing access controls on APIs
Рђб Privilege escalation

Example: /api/users/123/profile Рєњ /api/users/456/profile

Рђб Sensitive data exposure
Рђб Weak encryption algorithms
Рђб Missing encryption in transit/at rest
Рђб Hardcoded secrets

Рђб SQL injection
Рђб NoSQL injection
Рђб OS command injection
Рђб LDAP injection

Рђб Missing security controls by design
Рђб No threat modeling
Рђб Insufficient business logic controls

Рђб Default credentials
Рђб Unnecessary features enabled
Рђб Verbose error messages
Рђб Missing security headers

Рђб Outdated libraries/frameworks
Рђб Unpatched dependencies

Рђб Weak password policies
Рђб Missing MFA
Рђб Session fixation
Рђб Credential stuffing vulnerability

When: Pre-production, periodic production scans
Finds: Runtime vulnerabilities, configuration issues

Tools:

Рђб OWASP ZAP
Рђб Qualys WAS
Рђб Rapid7 InsightAppSec
Рђб Invicti (Netsparker)
